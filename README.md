# ✦ Elara Voss — Vintage Portfolio

A dark academia personal portfolio website with vintage aesthetics, floating botanical particles, moon/star intro screen, and responsive design across all pages.

## 📁 File Structure

```
vintage-portfolio/
├── index.html          ← Homepage (moon intro + hero + about + work preview)
├── css/
│   └── style.css       ← All shared styles
├── js/
│   └── main.js         ← All shared JavaScript (nav, lightbox, scroll reveal, particles)
└── pages/
    ├── about.html      ← About page (story, skills, timeline, photo frames)
    ├── work.html       ← Work/Projects page (filterable project cards)
    ├── gallery.html    ← Photo gallery (masonry grid, lightbox, filter by category)
    ├── journal.html    ← Blog/Journal page (article cards + sidebar)
    └── contact.html    ← Contact page (form + social links)
```

## 🚀 Deploy to GitHub Pages

1. Create a new GitHub repository named `portfolio` (or any name)
2. Upload all files **keeping the folder structure intact**
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch and `/ (root)` folder
5. Click **Save** — your site will be live at:
   `https://yourusername.github.io/portfolio/`

## ✨ Features

- 🌙 **Moon & Stars intro screen** — fades automatically or on button click
- 🍂 **Floating botanical particles** — leaves, butterflies, vintage symbols
- 📜 **Torn paper SVG dividers** between every section
- 🖼️ **Lightbox gallery** — click any photo to zoom (About, Gallery, Contact)
- 🔍 **Filter tabs** — Work page (by category), Gallery page (by subject)
- 📱 **Fully responsive** — hamburger menu, fluid grids, mobile-friendly buttons
- ✦ **Scroll reveal animations** — elements fade in as you scroll
- 📬 **Contact form** — with project type, budget, and message fields
- 🎨 **Vintage photo treatment** — sepia filter + aged effects on all images

## 🎨 Customise

Replace placeholder text in each `.html` file:
- `Elara Voss` → your name
- `hello@elaravoss.com` → your email
- `yourusername` in GitHub links → your GitHub username
- Project descriptions, journal articles → your real content
- Unsplash photo URLs → your own photos
